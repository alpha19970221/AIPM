# 我做过的产品

## 时光碎记 IOSAPP

1. 目标用户是所有有记录习惯的人
2. 目标用户原来会打开多个app来记录不同的事，如健身饮食，生理期，旅行计划等。
3. 原来的流程用户需要打开多个app，来回切换很不方便。并且用户需要手动使用很多按钮功能。
4. 问题随着手机越用越慢会变得越来越明显，打开app的时间成本变高了，甚至app的流畅度也变差，影响用户体验。
5. AI介入后用户可以采用对话式来记录自己想要记录的东西，AI会在后台把用户要记录的内容提取出来，并且将其自动分类到我们设计好的模块中。这样用户既不用打开多个app去记录，又不用手动去繁琐的添加记录各种东西，让记录变得更加方便。
6. 投入市场进行测试
